## üìÑ About
  SELY (SentimEnt anaLYsis) is a Twitter bot that captures a call and returns the sentimental analysis of the user's `x` tweets (limited by API). The entire analysis is done by Machine Learning (ML) based on the Robustly Optimized BERT Pretraining Approach model ([RoBERTa](https://pytorch.org/hub/pytorch_fairseq_roberta/)).
 `accuracy: 83.61%`
